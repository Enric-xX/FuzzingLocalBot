🛠️ Instalación

# Clona o descarga el repositorio
git clone https://github.com/tu-usuario/fuzzing-bot.git
cd fuzzing-bot

# Instala las dependencias de Python
pip install -r requirements.txt

# Ejecuta el bot (necesitas AutoHotkey instalado)
fuzzing_bot.ahk

📌 Descarga AutoHotkey aquí

🚀 Uso

  1. Abre el bot → Ejecuta fuzzing_bot.ahk.
  2. Selecciona navegador en el menú desplegable.
  3. Introduce el dominio (ej: https://ejemplo.com).
  4. Presiona "INICIAR FUZZING".
  5. No toques el teclado durante el proceso.
  6. Los resultados se guardarán como fuzz_results_[fecha].txt y .json.
