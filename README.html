📖 Descripción

El Fuzzing Bot es una herramienta de automatización para pruebas de penetración que permite fuzzear directorios y extensiones en un dominio objetivo.

Combina AutoHotkey para simular interacción humana en el navegador y Python para realizar verificaciones rápidas y en paralelo.

⚡ Características

  • Carga automática de `extensiones.txt` con miles de rutas.
  • Simulación humana en el navegador (escritura por teclado).
  • Verificación en paralelo con Python (50 threads).
  • Resultados en TXT y JSON.
  • Interfaz gráfica para seleccionar navegador y dominio.
  • Aviso de VPN para proteger tu privacidad.
  • Progreso en tiempo real durante la ejecución.

📂 Estructura del proyecto

📁 fuzzing_bot/
📄 fuzzing_bot.ahk ← Bot principal (500+ líneas)
🐍 fuzz_backend.py ← Backend en Python (500+ líneas)
📄 extensiones.txt ← Diccionario de directorios y extensiones
📄 README.html ← Esta página

🛠️ Requisitos e instalación

Requisitos

  • Windows (para AutoHotkey)
  • Python 3.8+ con la librería requests
  • Navegador (Chrome, Edge, Firefox, Brave, Opera)

Instalación

  • Descarga y extrae el ZIP en una carpeta.
  • Asegúrate de que extensiones.txt esté en la misma carpeta.
  • Ejecuta fuzzing_bot.ahk (necesitas AutoHotkey instalado).
  • El backend Python se ejecutará automáticamente.

🚀 Uso

  1. Abre el bot (ejecuta fuzzing_bot.ahk).
  2. Selecciona un navegador en el menú desplegable.
  3. Introduce el dominio que quieres fuzzear (ej: https://ejemplo.com).
  4. Presiona "INICIAR FUZZING".
  5. No toques el teclado durante el proceso.
  6. Los resultados se guardarán como fuzz_results_[fecha].txt y .json.
